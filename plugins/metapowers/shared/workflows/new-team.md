# Set Up a New Team

Build and launch a new team with leadership structure and processes.

## Steps

1. `/leadership:assess <team>` — Assess leadership style and team landscape
2. `/leadership:vision <team>` — Define team purpose, values, and culture
3. `/leadership:build <team>` — Structure roles, responsibilities, delegation
4. `/leadership:develop <team>` — Create growth plans and set OKRs
5. `/leadership:sustain <team>` — Establish rituals (1:1s, retros, reviews)
6. `/project-management:project-charter <project>` — Define first project
7. `/project-management:team-setup <project>` — Set up Scrum ceremonies
8. `/project-management:sprint-planning <project>` — Plan first sprint
9. `/development:plan <first-feature>` — Plan the team's first deliverable
10. `/leadership:one-on-one <team-member>` — Prepare first 1:1s

## Tips
- Complete leadership phases before project management
- Run individual 1:1 prep for each team member
- Revisit leadership:assess quarterly
