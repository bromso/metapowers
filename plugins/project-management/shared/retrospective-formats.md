# Retrospective Formats

A collection of proven retrospective formats. Pick the one that best suits your team's current needs.

---

## 1. Start / Stop / Continue

> **Best for:** General-purpose retrospectives, new teams.
> **Time:** 45-60 minutes.

Divide feedback into three categories:

| Category     | Prompt                                                        |
| ------------ | ------------------------------------------------------------- |
| **Start**    | What should we begin doing that we are not doing today?       |
| **Stop**     | What should we stop doing because it is not helping?          |
| **Continue** | What is working well and should carry on into the next Sprint?|

### Suggested Time-boxes

1. **Silent brainstorm** (5 min) — Everyone writes sticky notes independently.
2. **Share & group** (10 min) — Read notes aloud and cluster related items.
3. **Dot vote** (5 min) — Each person gets 3 votes to surface top items.
4. **Discuss top items** (20-30 min) — Dig into the highest-voted topics.
5. **Commit to actions** (5 min) — Agree on 1-3 concrete action items with owners.

## 2. 4Ls: Liked / Learned / Lacked / Longed For

> **Best for:** Balanced reflection covering positives and gaps.
> **Time:** 60 minutes.

| Category        | Prompt                                              |
| --------------- | --------------------------------------------------- |
| **Liked**       | What did you enjoy or appreciate this Sprint?       |
| **Learned**     | What new knowledge or insight did you gain?         |
| **Lacked**      | What was missing — tools, information, support?     |
| **Longed For**  | What do you wish you had that would make things better? |

### How to Run

1. Draw four quadrants on a board (physical or virtual) labelled with each L.
2. Silent brainstorm (7 min).
3. Walk through each quadrant, grouping and discussing (35 min).
4. Identify themes and agree on action items (15 min).

## 3. Sailboat

> **Best for:** Visual thinkers, teams that enjoy metaphor-driven discussion.
> **Time:** 45-60 minutes.

Draw a sailboat on water with four elements:

| Element               | Represents                                          |
| --------------------- | --------------------------------------------------- |
| **Island** (Goal)     | Where we want to be — our Sprint or project goal.   |
| **Wind** (Tailwinds)  | Forces pushing us forward — what is helping?        |
| **Anchor** (Drag)     | Forces holding us back — what is slowing us down?   |
| **Rocks** (Risks)     | Hidden dangers ahead — what could go wrong?         |

### How to Run

1. Draw the sailboat visual on a shared board.
2. Silent brainstorm (7 min) — sticky notes on each element.
3. Discuss each element starting with Wind, then Anchor, Rocks, and finally Island (30-40 min).
4. Convert Anchors and Rocks into action items (10 min).

## 4. Mad / Sad / Glad

> **Best for:** Quick emotional check-in, surfacing team morale.
> **Time:** 30-45 minutes.

| Category  | Prompt                                                     |
| --------- | ---------------------------------------------------------- |
| **Mad**   | What frustrated or angered you this Sprint?                |
| **Sad**   | What disappointed you or made you feel let down?           |
| **Glad**  | What made you happy, proud, or grateful?                   |

### How to Run

1. Create three columns: Mad, Sad, Glad.
2. Silent brainstorm (5 min).
3. Share and group (10 min).
4. Discuss patterns and vote on priority items (10-15 min).
5. Define action items (5 min).

---

## 5. Timeline Retrospective

> **Best for:** Long Sprints, multi-Sprint reviews, post-mortems.
> **Time:** 60-90 minutes.

The team reconstructs the Sprint (or project phase) chronologically and annotates key events.

### Steps

1. **Draw the timeline** (5 min) — A horizontal line representing the Sprint, with start and end dates marked.
2. **Plot events** (10 min) — Each person adds significant events (deployments, blockers, breakthroughs, incidents) to the timeline.
3. **Annotate mood** (5 min) — Place an energy/mood indicator (high or low) at each event.
4. **Walk the timeline** (25-40 min) — Move left to right, discussing each event cluster: What happened? Why? What was the impact?
5. **Extract lessons** (10 min) — Identify recurring patterns or systemic issues.
6. **Agree on actions** (5-10 min) — Commit to 1-3 improvements for the next Sprint.

## Facilitation Tips

Regardless of format, keep these principles in mind:

- **Time-box everything.** Respect the clock so every section gets attention.
- **Silent brainstorm first.** Let everyone think independently before group discussion to avoid anchoring bias.
- **Dot voting.** Give each person 3 votes to prioritise topics — this keeps discussion focused.
- **Limit action items.** Commit to 1-3 actions per retrospective. More than that rarely gets done.
- **Follow up.** Review last retro's action items at the start of the next retrospective.
- **Rotate the facilitator.** Sharing facilitation gives everyone ownership and fresh perspectives.
- **Safe space.** Remind the team that the retrospective is blame-free. Focus on systems and processes, not individuals.
