## Key Ratios Reference

| Category | Ratio | Formula |
|----------|-------|---------|
| **Liquidity** | Current Ratio | Current Assets / Current Liabilities |
| | Quick Ratio | (Cash + Receivables) / Current Liabilities |
| **Profitability** | Gross Margin | (Revenue - COGS) / Revenue |
| | Net Margin | Net Income / Revenue |
| | EBITDA Margin | EBITDA / Revenue |
| **Efficiency** | CAC Payback | CAC / Monthly Gross Profit per Customer |
| | LTV:CAC | Lifetime Value / Customer Acquisition Cost |
| | Revenue per Employee | Annual Revenue / Headcount |
| **Leverage** | Debt-to-Equity | Total Debt / Total Equity |
| | Interest Coverage | EBITDA / Interest Expense |
