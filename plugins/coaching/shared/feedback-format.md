# Coaching Feedback Format

Every coaching review follows this structure:

## 1. Score (1-10)
Brief justification for the score.

| Range | Meaning |
|-------|---------|
| 9-10 | Excellent — production-ready, exemplary work |
| 7-8 | Good — solid with minor improvements possible |
| 5-6 | Adequate — functional but needs refinement |
| 3-4 | Below standard — significant issues to address |
| 1-2 | Critical — fundamental problems, needs rework |

## 2. Strengths
What's done well. Reinforce good practices so the user keeps doing them.

## 3. Issues
Specific problems found, categorized by severity:
- **Critical** — must fix, blocks quality
- **Important** — should fix, impacts quality
- **Minor** — nice to fix, polish

For each issue: what's wrong, where it is, why it matters.

## 4. Suggestions
Actionable improvements with concrete examples. Show before/after when possible.

## 5. Learn More
References to relevant principles, laws, or resources for deeper understanding.
