## Purpose

The Review phase examines drafted or existing legal documents for risks, problematic clauses, and negotiation opportunities. It ensures documents protect your interests before execution.

## Skills

### Contract Review

```
/legal:contract-review <topic>
```

Performs a comprehensive contract review. Evaluates terms, identifies imbalances, checks for missing provisions, and assesses overall risk level.

**Output:** Review summary, risk assessment, recommended changes → `.metapowers/legal/<topic>/02-review.md`

### Clause Analysis

```
/legal:clause-analysis <topic>
```

Deep-dives into specific contract clauses. Analyzes enforceability, market standards, and potential implications of individual provisions.

**Output:** Clause-by-clause analysis with market comparisons → `.metapowers/legal/<topic>/02-review.md`

### Red Flag Detection

```
/legal:red-flag-detection <topic>
```

Scans contracts for high-risk provisions — unlimited liability, one-sided indemnification, broad IP assignments, auto-renewal traps, and other problematic terms.

**Output:** Red flag report with severity ratings and remediation suggestions → `.metapowers/legal/<topic>/02-review.md`

### Negotiation Prep

```
/legal:negotiation-prep <topic>
```

Prepares negotiation strategies for contract discussions. Identifies leverage points, fallback positions, and priority terms to negotiate.

**Output:** Negotiation playbook with positions and fallbacks → `.metapowers/legal/<topic>/02-review.md`

### IP Review

```
/legal:ip-review <topic>
```

Reviews intellectual property provisions in contracts. Analyzes IP ownership, licensing terms, assignment clauses, and work-for-hire provisions.

**Output:** IP analysis with ownership mapping and risk assessment → `.metapowers/legal/<topic>/02-review.md`

### Compliance Review

```
/legal:compliance-review <topic>
```

Reviews documents against applicable regulatory requirements. Checks for regulatory compliance gaps and recommends necessary additions.

**Output:** Compliance gap analysis with regulatory mapping → `.metapowers/legal/<topic>/02-review.md`
