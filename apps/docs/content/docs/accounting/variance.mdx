## Purpose

The Variance utility compares actual financial results against budgeted or expected amounts, identifying favorable and unfavorable variances and their root causes.

## Usage

```
/accounting:variance <project>
```

## What It Does

1. Reads actual results from the financial statements
2. Compares against budget or prior period data
3. Calculates dollar and percentage variances by account
4. Classifies variances as favorable or unfavorable
5. Identifies significant variances requiring investigation

## Output

- Variance report by account with dollar and percentage differences
- Favorable/unfavorable classification
- Significant variance highlights with suggested root causes
