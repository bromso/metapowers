## Purpose

The Audit utility performs detailed reviews of specific accounts or transaction classes, applying substantive testing procedures to verify accuracy, completeness, and validity of recorded amounts.

## Usage

```
/accounting:audit <project>
```

## What It Does

1. Selects accounts or transaction classes for review
2. Designs audit procedures (vouching, tracing, analytical review)
3. Tests sample transactions against source documents
4. Evaluates account balances for reasonableness
5. Documents findings and exceptions

## Output

- Audit program with procedures and scope
- Test results with exceptions noted
- Findings summary with recommendations
