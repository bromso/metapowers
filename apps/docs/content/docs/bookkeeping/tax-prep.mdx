## Purpose

The Tax Prep utility organizes financial records into the categories and formats required for tax filing, ensuring completeness and identifying potential deductions or issues before submission.

## Usage

```
/bookkeeping:tax-prep <project>
```

## What It Does

1. Reviews categorized transactions for tax-relevant classifications
2. Groups deductible expenses by tax category
3. Identifies missing documentation or receipts
4. Summarizes revenue, expenses, and estimated tax liability
5. Flags items requiring accountant review

## Output

- Tax category summary with totals
- Missing documentation checklist
- Estimated tax liability overview
- Items flagged for professional review
