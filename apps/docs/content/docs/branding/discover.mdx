## Purpose

The Discover phase gathers insights about the business, audience, competitors, and market. It asks key questions and conducts research to build a foundation for brand strategy.

## Usage

```
/branding:discover <brand-name>
```

**Example:**
```
/branding:discover acme
```

## What It Does

1. Asks discovery questions about the business, audience, goals, and aspirations
2. Audits existing brand assets if they exist (logo, colors, messaging)
3. Researches competitor brands via web search
4. Builds audience personas with needs and pain points
5. Writes findings to `.branding/<brand>/01-discover.md`

## Output

Creates `.branding/<brand>/01-discover.md` containing:

- Business overview and goals
- Existing brand audit (if applicable)
- Competitive landscape analysis
- Target audience personas
- Market insights and opportunities
