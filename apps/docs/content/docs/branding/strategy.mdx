## Purpose

The Strategy phase synthesizes discovery findings into a clear brand platform — positioning, values, personality, and differentiation that guide all brand decisions.

## Usage

```
/branding:strategy <brand-name>
```

**Example:**
```
/branding:strategy acme
```

## Prerequisites

Requires `.metapowers/branding/<brand>/01-discover.md` from the Discover phase. Use `--skip-checks` to bypass.

## What It Does

1. Defines brand purpose, vision, and mission
2. Establishes 3-5 core values with behavioral descriptions
3. Crafts a positioning statement
4. Defines brand personality traits
5. Maps competitive differentiation
6. Writes the brand platform to `.metapowers/branding/<brand>/02-strategy.md`

## Output

Creates `.metapowers/branding/<brand>/02-strategy.md` containing:

- Purpose, vision, and mission
- Core values
- Positioning statement
- Brand personality traits
- Competitive differentiation map
- Target audience profiles

## Next Phase

After Strategy, proceed to [Verbal](/docs/branding/verbal) to develop the brand's voice and messaging.
