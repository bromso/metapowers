## Purpose

The Assess phase gathers, consolidates, and validates financial data from across the organization, establishing a reliable foundation for planning, forecasting, and budgeting.

## Usage

```
/finance:assess <project>
```

## What It Does

1. Collects financial data from bookkeeping and accounting outputs
2. Validates data completeness and consistency
3. Normalizes formats and reconciles discrepancies
4. Summarizes current financial position and trends
5. Writes the assessment to `.metapowers/finance/<project>/01-assess.md`

## Output

Creates `.metapowers/finance/<project>/01-assess.md` containing:

- Consolidated financial data summary
- Data quality assessment and gaps identified
- Current financial position overview
- Historical trend analysis

## Next Phase

After assessment, proceed to [Plan](/docs/finance/plan) to define strategic objectives and KPIs.
