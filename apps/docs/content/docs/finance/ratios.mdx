## Purpose

The Ratios utility calculates standard financial ratios to assess the organization's liquidity, profitability, efficiency, and leverage, providing a snapshot of financial health.

## Usage

```
/finance:ratios <project>
```

## What It Does

1. Calculates liquidity ratios (current ratio, quick ratio, cash ratio)
2. Computes profitability ratios (gross margin, operating margin, ROE, ROA)
3. Analyzes efficiency ratios (asset turnover, inventory turnover, DSO)
4. Evaluates leverage ratios (debt-to-equity, interest coverage, debt-to-assets)
5. Benchmarks against industry standards where available

## Output

- Ratio dashboard organized by category
- Period-over-period trend analysis
- Industry benchmark comparison
- Flagged ratios outside healthy ranges
