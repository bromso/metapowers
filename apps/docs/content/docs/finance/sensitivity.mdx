## Purpose

The Sensitivity utility tests how changes in individual input variables (price, volume, cost, growth rate) affect key financial outputs, identifying which variables have the greatest impact on results.

## Usage

```
/finance:sensitivity <project>
```

## What It Does

1. Identifies key input variables in the financial model
2. Varies each input independently across a defined range
3. Measures the impact on target outputs (revenue, profit, valuation)
4. Ranks variables by sensitivity (highest impact first)
5. Produces tornado charts and data tables

## Output

- Sensitivity matrix with input ranges and output impacts
- Variable ranking by impact magnitude
- Tornado chart data for visualization
