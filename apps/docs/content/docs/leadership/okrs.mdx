## Purpose

Create or review OKRs — clear objectives with measurable key results aligned to team and organizational goals.

## Usage

```
/leadership:okrs <scope>
```

**Example:**
```
/leadership:okrs Q3 engineering team
```

## What It Does

1. Defines 2-3 objectives (qualitative, inspirational, time-bound)
2. Sets 3-5 measurable key results per objective (outcomes, not activities)
3. Aligns upward to organizational goals
4. Validates capacity and identifies what to deprioritize
5. Writes to `.leadership/<topic>/okrs.md`
