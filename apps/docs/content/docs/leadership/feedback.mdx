## Purpose

Prepare structured feedback using the SBI (Situation, Behavior, Impact) model and the CFR (Conversations, Feedback, Recognition) framework.

## Usage

```
/leadership:feedback <situation>
```

**Example:**
```
/leadership:feedback missed deadline on API migration
```

## What It Does

1. Structures feedback using SBI: specific situation, observable behavior, concrete impact
2. Adds feedforward guidance (what to do differently)
3. Prepares delivery notes with tone guidance and anticipated reactions
4. Writes to `.leadership/<topic>/feedback.md`
