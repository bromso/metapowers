## Purpose

Work through a decision using the appropriate framework — decide alone, consult, delegate, or seek consensus based on impact and urgency.

## Usage

```
/leadership:decision <decision-topic>
```

**Example:**
```
/leadership:decision rewrite auth system or patch it
```

## What It Does

1. Determines the right decision mode (impact/urgency matrix)
2. Analyzes options with pros, cons, risks, and reversibility
3. Makes a recommendation with monitoring plan and rollback strategy
4. Writes to `.leadership/<topic>/decision.md`
