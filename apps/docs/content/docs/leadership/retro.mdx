## Purpose

Facilitate a structured team retrospective using the Start/Stop/Continue format with root cause analysis and actionable outcomes.

## Usage

```
/leadership:retro <period-or-project>
```

**Example:**
```
/leadership:retro sprint 12
```

## What It Does

1. Structures reflection: Start (new practices), Stop (drop), Continue (keep)
2. Identifies root causes and recurring patterns
3. Converts insights into 2-3 assignable action items with owners and deadlines
4. Writes to `.metapowers/leadership/<topic>/retro.md`
