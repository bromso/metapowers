## Purpose

Prepare a structured 1:1 meeting agenda with talking points, feedback to deliver, and action items to follow up on.

## Usage

```
/leadership:one-on-one <person-or-topic>
```

**Example:**
```
/leadership:one-on-one weekly with Sarah
```

## What It Does

1. Prepares an agenda following the 1:1 template (check-in, their topics, progress, growth, actions)
2. Suggests conversation starters based on context
3. Identifies feedback to deliver and growth goals to check in on
4. Writes to `.metapowers/leadership/<topic>/one-on-one.md`
